# cpp-whatisthisgame

This is a copy of my game
[whatisthisgame](http://github.com/crockeo/whatisthisgame) written in C++
instead of Haskell.

### Building

Assuming you have CMake installed (with the minimum version of 2.8):

```bash
$ cmake .
$ make
$ ./build/bin/whatisthisgame
```
